# XOR解密工具

一个基于浏览器的XOR解密工具，支持多种输入格式和密钥类型，完全在本地运行，无需服务器。

## 功能特性

- 多种密钥类型支持：文本、十六进制、二进制、十进制
- 多种输入格式：十六进制、Plain Text、Base64、Binary、文件上传
- 自动检测解密后的文件类型（支持图片、文档、压缩包、文本等）
- **图片预览功能**：解密后的图片（JPEG、PNG、GIF）可直接在页面中显示，无需下载
- **文本预览功能**：支持中文和英文文本的实时预览
- 一键保存解密结果到本地
- 纯前端实现，数据不上传服务器，安全可靠

## 使用方法

### 在线访问

项目在线地址：https://xor.bbd0e568.er.aliyun-esa.net/

### 1. 打开工具

直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 设置密钥

- 在"密钥设置"区域输入解密密钥
- 选择密钥类型：
  - **Text (文本)**: 直接输入文本密钥
  - **Hex (十六进制)**: 输入十六进制格式的密钥
  - **Binary (二进制)**: 输入二进制字符串格式的密钥
  - **Decimal (十进制)**: 输入0-255之间的十进制数值（用空格或逗号分隔）

### 3. 输入加密数据

选择以下任一方式输入需要解密的数据：

- **十六进制 (HEX)**: 输入十六进制编码的加密数据
- **Plain Text**: 输入纯文本格式的加密数据
- **Base64**: 输入Base64编码的加密数据
- **Binary**: 输入二进制字符串格式的加密数据
- **文件上传**: 选择加密文件，并指定文件内容格式

### 4. 解密

点击"解密"按钮进行解密操作。

### 5. 查看和保存结果

解密成功后，页面会显示解密结果区域，包含以下信息：

- **文件信息**：显示解密后文件的大小和类型
- **预览区域**：
  - **图片文件**：JPEG、PNG、GIF 格式的图片会直接在页面中显示，您可以立即查看解密后的图片效果
  - **文本文件**：包括中文和英文文本，会以格式化的方式显示在预览区域中
  - **其他文件**：二进制文件（如 PDF、ZIP 等）会提示无法预览，需要下载后查看
- **保存文件**：点击"保存文件"按钮，将解密后的文件下载到本地

#### 预览功能说明

- **图片预览**：工具会自动识别图片格式，解密成功后直接在浏览器中渲染显示，支持缩放和查看
- **文本预览**：支持 UTF-8 编码的文本，包括中文、英文等，最多显示前 5000 个字符
- **文件保存**：所有类型的文件都可以通过"保存文件"按钮下载到本地，文件扩展名会根据检测到的文件类型自动设置

## 支持的文件类型

工具会自动检测解密后的文件类型，支持：

- 图片：JPEG、PNG、GIF
- 文档：PDF
- 压缩包：ZIP、GZIP
- 文本：纯文本文件
- 其他：二进制文件

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 无需任何依赖或框架
- 使用浏览器原生 API（TextEncoder、TextDecoder、Blob、File API）

- 建议使用现代浏览器（Chrome、Firefox、Edge等）以获得最佳体验
